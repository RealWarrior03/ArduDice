# ArduDice

The open-source Arduino-based digital dice

## Features (Version 1.0.1)
- various dice
  - d4
  - d6
  - d8
  - d10
  - d12
  - d20
- up to 5 dice at once
- (running) total

## Roadmap
- shake to throw (new sensor needed)
- custom enclosure (3d printed)


## Bill of materials
- Arduino Nano
- 0.91" I2C OLED Display
- buttons
- breadboard
- jumper wires
